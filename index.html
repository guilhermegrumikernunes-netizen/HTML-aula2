historia da criação do html

No início da década de 1990, um cientista chamado Tim Berners-Lee trabalhava no CERN, um grande centro de pesquisas na Europa. Ele precisava de uma forma simples de compartilhar documentos entre pesquisadores que estavam em diferentes computadores.

Tim Berners-Lee

Para resolver esse problema, ele criou uma linguagem simples chamada HTML, que significa HyperText Markup Language. Essa linguagem permite estruturar documentos utilizando elementos chamados de tags.

HyperText Markup Language
Essa linguagem se tornou a base da criação de páginas na internet.
